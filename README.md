# Flask E-commerce

## Description
Ce projet est une application web de vente en ligne réalisée avec Flask, un framework Python simple et léger.
Il a été créé dans le but d’apprendre les bases du développement web backend et de comprendre comment fonctionne une application web.

## Objectifs 
L’application permet :
- d’afficher des produits
- de les organiser par catégories
- d’utiliser une base de données pour stocker les informations
- de naviguer entre plusieurs pages grâce aux routes Flask

Ce projet m’aide à comprendre :
- comment fonctionne Flask
- comment relier Python, HTML et une base de données
- comment structurer un projet web de manière propre
- comment utiliser des templates pour afficher des pages dynamiques

## Conclusion 
Ce dépôt est destiné à l’apprentissage et pourra évoluer progressivement avec de nouvelles fonctionnalités.

## Technologies
- Flask
- SQLAlchemy
- SQLite 

## Installation
pip install -r requirements.txt

## Lancer l'app
python run.py
